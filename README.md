# SUBMERGED — AI-Based Pathfinding Game

An interactive grid-based game where the player navigates through a flood-affected environment while an AI system computes the optimal escape path using the A* algorithm.

* Overview

"Submerged" simulates a disaster scenario where the goal is to reach a safe zone while avoiding flooded areas.

The game combines real-time gameplay with AI-driven decision-making, allowing players to compare their path with the optimal path generated by the A* algorithm.

* AI Concept Used

 A* Pathfinding Algorithm

The game uses the A* algorithm to:
- Compute the shortest path to the goal
- Avoid flooded and blocked cells
- Minimize traversal cost through shallow water

Heuristic used:
- Manhattan Distance

Cost system:
- Normal cell = 1
- Shallow water = higher cost
- Flooded/blocked cells = not traversable

* Features

- Multiple maps 
- AI-generated hint system showing optimal path
- Utility-based scoring system
- Player movement using keyboard or on-screen controls
- Performance grading system (S, A, B, C, F)

* Scoring System

Score = 1000  
        - (extra moves × 15)  
        - (shallow crossings × 25)  
        - (hints used × 30)  
        + optimal path bonus  

Efficient navigation results in higher scores, while unnecessary moves and hints reduce performance.

* Objective

Reach the safe zone(s) while:
- Avoiding flooded cells
- Minimizing shallow water crossings
- Taking the shortest possible path
